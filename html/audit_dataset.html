audit_dataset module¶

audit_dataset.py — Analyse et visualisation du dataset AeroPredict¶

Produit des rapports PDF individuels pour chaque analyse :

  • Distribution par famille de profils

  • Distribution des features géométriques

Usage :

python audit_dataset.py python audit_dataset.py –input mon_dataset.csv –output_dir rapport_pdf/

audit_dataset.audit_dataset(filepath='dataset.csv', output_dir='audit_reports')¶

Audite le dataset XFoil et génère les rapports PDF correspondants.

Charge le CSV, affiche un résumé qualité dans la console (doublons, valeurs manquantes, profils incomplets), puis génère les trois rapports PDF (familles, features géométriques, scatter plots) dans output_dir via report_familles(), report_features() et report_scatter().

Paramètres:

  • filepath (str) – Chemin du fichier CSV du dataset à auditer.

  • output_dir (str) – Dossier de sortie des rapports PDF (nettoyé des anciens PDF avant génération, voir clean_output_dir()).

Renvoie:

Rien.

Type renvoyé:

None

audit_dataset.clean_output_dir(output_dir)¶

Nettoie les anciens fichiers PDF du dossier sans supprimer le dossier lui-même.

Paramètres:

output_dir (str) – Chemin du dossier de sortie à nettoyer (créé s’il n’existe pas).

Renvoie:

Rien.

Type renvoyé:

None

audit_dataset.load(filepath)¶

Charge le dataset CSV et dérive la colonne famille à partir du nom NACA/UIUC.

Paramètres:

filepath (str) – Chemin du fichier CSV à charger.

Renvoie:

DataFrame du dataset, enrichi d’une colonne famille (lettres initiales du nom de profil, en minuscules).

Type renvoyé:

pd.DataFrame

audit_dataset.report_familles(output_dir, df)¶

Génère un PDF avec les graphiques des familles de profils.

Paramètres:

  • output_dir (str) – Dossier de sortie où écrire le PDF généré.

  • df (pd.DataFrame) – DataFrame du dataset, doit contenir une colonne famille.

Renvoie:

Rien. Le PDF 01_familles_profils.pdf est écrit sur disque.

Type renvoyé:

None

audit_dataset.report_features(output_dir, df)¶

Génère un PDF avec la distribution des 8 features géométriques.

Paramètres:

  • output_dir (str) – Dossier de sortie où écrire le PDF généré.

  • df (pd.DataFrame) – DataFrame du dataset, doit contenir les colonnes géométriques (t, camber, x_t, x_c, LE_radius, TE_angle, t_over_xt, area) et naca.

Renvoie:

Rien. Le PDF 02_features_geometriques.pdf est écrit sur disque.

Type renvoyé:

None

audit_dataset.report_scatter(output_dir, df)¶

Génère un PDF avec les scatter plots entre features géométriques, par source.

Paramètres:

  • output_dir (str) – Dossier de sortie où écrire le PDF généré.

  • df (pd.DataFrame) – DataFrame du dataset, doit contenir les colonnes géométriques et une colonne source (naca_grid ou uiuc).

Renvoie:

Rien. Le PDF 03_scatter_plots.pdf est écrit sur disque.

Type renvoyé:

None
